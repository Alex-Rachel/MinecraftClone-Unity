## 特性

* 无限地形生成
* 洞穴和矿物
* 光照与物理
* 能爆炸的TNT
* 能流动的水
* 受重力影响的沙子
* 世界会被保存到文件中
* 音效和粒子特效
* 一些bug (〃'▽'〃)
